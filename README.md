## How to host on GitHub Pages

1. Create a repository named exactly `rohit-kumar-j.github.io` on GitHub
2. Push this folder to the `main` branch
3. Go to **Settings → Pages → Source → Deploy from branch → main / root**
4. Your site will be live at `https://rohit-kumar-j.github.io` within a minute

> Your CV (`files/cv.pdf`) will be publicly accessible at `https://rohit-kumar-j.github.io/files/cv.pdf` — yes, GitHub Pages automatically hosts everything you push.
